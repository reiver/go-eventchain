# go-eventchain

Package `eventchain` provides basic building blocks for doing **Event Sourcing** and **Log-Driven Development**,
for the Go programming language.

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-eventchain
